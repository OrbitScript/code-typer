# ⚡ CODE-TYPER Quick Start Guide

Get typing in 2 minutes!

## 🚀 Installation

```bash
# Clone repository
git clone https://github.com/OrbitScript/code-typer.git
cd code-typer

# Install dependencies
pip install -r requirements.txt
```

## 🎬 First Recording

### 1. Launch the App
```bash
python code_typer.py
```

### 2. Load Example Code
- Click **"📂 Load File"**
- Select `example_arduino.ino`
- Or paste your own code!

### 3. Set Your Speed
- **80 WPM** = Comfortable tutorial pace
- **2% errors** = Realistic typos

### 4. Record Your Video
1. Start your screen recorder (OBS, QuickTime, etc.)
2. Open Arduino IDE (or any text editor)
3. Click in the editor where you want typing to start
4. Press **F9** in CODE-TYPER
5. You have 5 seconds to switch windows!
6. Watch it type automatically! ✨

## ⌨️ Hotkeys

- **F9** - Start typing (5 sec countdown)
- **F10** - Pause/Resume
- **F11** - Stop immediately

## 💡 Pro Tips

### For Arduino Tutorials:
1. Open Arduino IDE
2. Create new sketch
3. Click in editor area
4. Load your .ino file in CODE-TYPER
5. Press F9 and explain while it types!

### For YouTube Videos:
- Use 60-80 WPM for teaching
- Enable realistic pauses
- 2% error rate looks natural

Write code → Load in CODE-TYPER → Start recording →

## ❓ Common Questions

**Q: Will it work with my editor?**  
A: Yes! Works with ANY text input (Arduino IDE, VS Code, Notepad, etc.)

**Q: Can I pause mid-typing?**  
A: Yes! Press F10 to pause/resume anytime

**Q: What if I make a mistake?**  
A: Press F11 to stop, fix it manually, then continue

**Q: Typing too fast/slow?**  
A: Adjust WPM in settings (20-200 range available)
